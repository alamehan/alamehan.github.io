Extension: Translator

  1. 1 Click Translator : Terjemahkan text cukup dengan select text tertentu. Mudah, cepat, dan fleksibel.
  2. XTranslate : Alternatif 1 Clik Translator. Google, Yandex, Bing. Translate Full Page, Speech, dll.
  3. Yandex : Translate full page. Original dan hasil terjemah side-by-side. Cocok untuk EPUB to HTML.
  4. Copyfish Free OCR Software : Convert image menjadi text yang dapat di copy/manipulasi/translate.
  5. Translator : Pop-up Translator, berfungsi juga untuk EPUB. Ada fitur save vocabulary (Dictionary).

Extension: Summarizer

  1. EndNote Click : Akses cepat sekali klik ke jutaan makalah penelitian. Bisa generate format PDF.
  2. Scholarcy : Ringkas makalah penelitian hanya dengan satu klik. Bisa membaca format PDF.
  3. GetDigest : Highlight informasi terpenting dari setiap artikel web secara cepat (Summary).
  4. Frase Article Summarizer : Ringkas artikel dalam sekali klik. Situs Medium.com, berita, dll.
  5. Crammer : Analisis text dalam web menggunakan AI. Temukan "Keypoint" dengan cepat.
  6. ‚á® Temukan beberapa Text Summarizing Tools terbaik lainnya yang tersedia. [Open]

Extension: Speech & More

  1. Natural Reader Text to Speech : Baca text sebagai audio dengan select text maupun full page.
  2. Read Aloud : Alternatif Natural Reader. Minim bug, text-to-speech di tracing dengan baik.
  3. Web & PDF Highlighter LINER : Highlight text penting dari website/PDF/EPUB anda.
  4. Mercury Reader : Bersihkan tampilan website, fokuskan hanya ke bagian tulisan/arikel saja.
  5. Unlimited Medium : Membuka akses artikel premium di Website Medium.com secara gratis.
  6. Wikiwand : Perbarui tampilan Wikipedia menjadi lebih ringan untuk dibaca. Good UX.
  7. Kindle Cloud Reader : Memungkinkan anda untuk membaca buku Kindle di web Browser.
  8. Side|Side : Buat 1 tab Chrome berisi 2 atau beberapa page yang berdampingan. Multitasking.
  9. Tab Resize Split Screen Layouts : Split/bagi layout windows Chrome menjadi 2, 3 atau 4.
